A ProseMirror-starter-app

This is a simple, minimal starter-app implementing the ProseMirror repo prosemirror-example-setup with the Javascript bundler RollUp.

This is purely a demonstration project, not a production grade implementation. For example, to demonstrate image support, we pull a little random image from an external source that can be slow to resolve sometimes.

So, for learning only. Not prod. Say that aloud. Once more.

Hello ProseMirror

This is editable text. You can focus it and start typing.

To apply styling, you can select a piece of text and manipulate its styling from the menu. The basic schema supports emphasis, strong text, links, code font, as well as images.

Block-level structure can be manipulated with key bindings (try ctrl-shift-2 to create a level 2 heading, or enter in an empty textblock to exit the parent block), or through the menu.

Try using the “list” item in the menu to wrap this paragraph in a numbered list.

Or grab the image (once it loads) and relocate it within the editor workspace.
